# AmDraCor
## Corpus description
The **Am**erican **Dra**ma **Cor**pus (beta) contains 40 plays by American playwrights, retrieved from various copyright-free online sources (see below for further details). Texts have been onboarded onto DraCor following the pipeline sketched in [BÃ¶rner et al. 2023](https://zenodo.org/record/7688632).

AmDraCor is maintained by [Daniil Skorinkin](https://www.uni-potsdam.de/de/digital-humanities/daniil-skorinkin) and [Luca Giovannini](https://lucagiovannini7.github.io).
